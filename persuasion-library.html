What It Is

The world's largest database of
what actually persuades people.

The Persuasion Library aggregates findings from select Rapid Message Tests run on the Grow Progress platform — a growing repository of findings across issues, audiences, and message types. Search by issue area, audience segment, message frame, or outcome.

See what's already worked for campaigns and organizations like yours, and what hasn't.
